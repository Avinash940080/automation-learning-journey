# Automation Learning Journey 

This repository documents my hands-on learning journey in workflow automation and no-code tools.

My goal is to learn and build real-world automations using:

* n8n
* Make.com
* Airtable
* APIs and Webhooks
* AI integrations

I am building automation projects, documenting my progress, and sharing my learnings through practical examples.

# Learning Goals

* Understand triggers, actions, and workflows
* Build automations using n8n and Make.com
* Design databases in Airtable
* Work with APIs, webhooks, and JSON data
* Document projects and maintain a portfolio
